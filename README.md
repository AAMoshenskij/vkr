# Система прогнозирования поведения протеза стопы из углепластика

Данный программный комплекс предназначен для идентификации упругих характеристик многослойного композитного пакета и прогнозирования его поведения при эксплуатационных нагрузках. Система разработана в рамках выпускной квалификационной работы и включает модули для послойного расчета, идентификации свойств материала и прогнозирования напряженно-деформированного состояния.

## Требования

- Python 3.8 или выше
- Библиотеки: `numpy`, `matplotlib`

Установка зависимостей:

```bash
pip install -r requirements.txt
```

## Инструкция по запуску

### 1. Идентификация упругих характеристик

Запуск идентификации:

```bash
python identification.py
```

В результате скрипт выведет оптимальные значения E1, E2, G12, построит карту ошибок и сохранит ее в файл `identification_slices.png`.

### 2. Прогнозирование поведения протеза

Перед запуском необходимо подставить идентифицированные значения в файл `predictor.py`:

```python
E1_IDENTIFIED = 25000  # МПа — результат из identification.py
E2_IDENTIFIED = 25000  # МПа
G12_IDENTIFIED = 1000  # МПа
```

Также настраиваются параметры пакета:

```python
N_LAYERS = 2                    # количество слоев
ANGLES_DEG = [90, 0]            # углы армирования
```

Запуск прогнозирования:

```bash
python predictor.py
```

Программа выполнит:

- Прогнозирование деформаций для экспериментальных нагрузок (30, 60, 90, 120, 135 кг)
- Расширенное прогнозирование для нагрузок (до 500 кг)
- Анализ прочности по критерию Цая-Ву для каждого слоя
- Определение критических нагрузок (зона внимания, критическое состояние, разрушение)
- Построение графиков зависимости индекса Цая-Ву от нагрузки

Результаты выводятся в консоль в виде таблиц и отчетов. Графики сохраняются в файлы:

- `layer_stresses.png` — распределение напряжений по слоям
- `extended_prediction.png` — зависимость индекса Цая-Ву и напряжений от нагрузки

## Принцип работы системы

1. **Идентификация** — методом перебора подбираются значения E1, E2, G12, минимизирующие отклонение расчетных деформаций от экспериментальных.

2. **Прогнозирование** — с использованием идентифицированных свойств рассчитываются деформации и напряжения в каждом слое для заданной нагрузки.

3. **Оценка прочности** — по критерию Цая-Ву вычисляется индекс разрушения для каждого слоя, на основе которого выдается вердикт о безопасности конструкции.
